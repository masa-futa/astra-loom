# リソースファイルをXcodeに追加

## 手順

1. **Xcodeでプロジェクトナビゲーターを開く**
   - 左側のサイドバーでファイル一覧を表示

2. **astra-loomフォルダを右クリック**
   - `Add Files to "astra-loom"...` を選択

3. **以下のファイルを選択**
   ```
   /Users/masakifutami/Documents/00_開発/astra-loom/apps/ios/astra-loom/astra-loom/stars.json
   /Users/masakifutami/Documents/00_開発/astra-loom/apps/ios/astra-loom/astra-loom/constellations.json
   ```

4. **オプションを設定**
   - ✅ `Copy items if needed` にチェック
   - ✅ `Create groups` を選択
   - ✅ `astra-loom` target にチェック

5. **Add をクリック**
